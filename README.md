# 🎨 UI/UX Case Study – Klik Indomaret App

## 📌 Overview
Revamp aplikasi klik indomaret untuk meningkatkan user experience dan kemudahan transaksi (kelompok 21 ui/ux design bootcamp myskill).

## 🎯 Problem
1. Pengguna tidak dapat membatalkan pesanan maupun melakukan komunikasi dengan toko maupun driver setelah melakukan pembayaran
User tidak dapat membatalkan pesanan yang sedang berlangsung
User tidak dapat chat dengan driver dan toko di aplikasi untuk mengkoreksi atau mengkonfirmasi pesanan
2. Pengguna tidak bisa melacak posisi driver secara real-time
User tidak bisa melihat ada update posisi petugas pengantar
